Papery Development Journey

Project Phases & Milestones

Phase 1: Planning & Research (4 weeks)

  • Researching and selecting an appropriate API; initially considered Unsplash but switched to Pexels due to API key generation issues
  • Choosing a theme for the website, found some good inspirations on Pinterest

Phase 2: Frontend Development (5 weeks)

  • Responsive layout implementation
  • Component-based structure development
  • Grid system creation
  • User friendly UI with slick animations

Phase 3: Backend Implementation (7 weeks)

  • Development of the favorites system
  • Implementation of search functionality
  • Creation of a filter system
  • Creation of a pagination system

Project Development Process

How did you organize the work?

My work organization followed these key principles:

  • Utilized Notion for efficient task management and progress tracking
  • Dedicated 2 hours daily to the project, ensuring consistent progress
  • Commenced with the frontend development, leveraging my expertise, and gradually delved into Go and JavaScript
  • Structuring the project was straightforward because our mentor provided a detailed PDF guide explaining everything needed to start and structure the project.

How did you manage time and set priorities?

Time management was structured around:

  • Dedicated 2 hours daily to the project, which significantly contributed to early completion
  • Prioritized core features initially, followed by enhancements; enjoyed the frontend phase while the backend posed more challenges

What was your documentation strategy?

Documentation was maintained throughout development:

  • README files for setup and contribution guidelines
  • Use of AI
